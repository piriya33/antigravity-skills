# Education Stack

A structured workflow for designing courses that stay aligned with learning outcomes, serve mixed-level audiences, and maintain consistency across sessions.

## Skills Overview

| Skill | Purpose | When to Use |
|-------|---------|-------------|
| **planning-courses** | Top-down course architecture | Starting a new course or restructuring |
| **designing-classes** | Individual session design | Designing each class |
| **aligning-curriculum** | Consistency & alignment check | After designing sessions |
